# 🌦️ Weather App

A simple and responsive Weather Application that allows users to search for any city and get real-time weather information using the OpenWeatherMap API.

---

## 📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature in Celsius
- 💧 Shows humidity level
- 🌬️ Displays wind speed
- 🌥️ Dynamic weather icons based on conditions
- ⚠️ Error handling for invalid city names
- ⌨️ Supports "Enter" key for quick search

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API
